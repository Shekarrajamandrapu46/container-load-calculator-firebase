Step 3: Execution

Skids (48″×40″), then sea containers — review details below after you calculate.

Standard 48″ × 40″ pallet — adjust to match your floor limit. Skids are packed supplier-by-supplier (never mixed on the same pallet). If a part has TY / QTY per pallet in the master, skid lines use that count (catalog rules). If TY is missing, each skid uses the tighter of this kg cap and the part’s L×W×H on a 48″×40″ deck (stack estimate). 40′ HC “volume” in results is driven by the same skid count (each skid = one nominal 48″×40″ slot × standard stack height), so container count follows palletized demand when QTY per pallet is set.

Exports need Step 2 vendor uploads for every supplier on the plan (see “Vendor Order Detail (per supplier)”). Summary PDF works after Calculate; PO column fills when vendor files are linked.
